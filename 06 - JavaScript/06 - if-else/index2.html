Operadores de comparação (condicionais)

Também conhecidos como operadores relacionais. Servem para formar as expressões condicionais para o comando if

Operadores:
  1. Igual(==) - Verifica se os valores comparados são iguais
  2. Idêntico(===) - Verifica se os valores comparados são iguais e do mesmo tipo
  3. Diferente(!=) - Verifica se os valores comparados são diferentes
  4. Não idêntico(!==) - Verifica se os valores comparados são diferentes e de tipos diferentes
  5. Menor(<) - Verifica se o da esquerda é menor que o valor da direita
  6. Maior(>) - Verifica se o da esquerda é maior que o valor da direita
  7. Menor igual(<=) - Verifica se o da esquerda é menor ou igual que o valor da direita
  8. Maior igual(>=) - Verifica se o da esquerda é maior ou igual que o valor da direita
